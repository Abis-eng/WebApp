Welcome to WebApp GitHub Actions CI/CD Demo!

This page is deployed automatically from the main branch using GitHub Actions.

About This Lab

This repository contains a Java Maven web application (JSP + Tomcat) with Docker and GitHub Actions pipelines.

Every time you push changes to main, workflows build the project and can deploy this site to GitHub Pages.

Run the Java application

Build and run with Docker locally:

mvn clean package -DskipTests
docker build -t webapp .
docker run -p 8081:8080 webapp

Then open: http://localhost:8081/WebApp/
